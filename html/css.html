CSS란?

CSS의 스타일 형식↓
1
선택자 { 속성명1: 속성값1; 속성명2: 속성값2; }
cs

· Cascading Style Sheet의 약자, Cascading즉, 절차적이다.
· 마크업이 실제 표시되는 방법을 기술하는 스타일 언어
· HTML, XHTML에서 주로 사용, XML에서도 사용 가능, W3C표준이다.
· '{'는 선언 시작 '}' 는 선언 끝을 나타낸다.
· 선택자에 해당하는 요소를 선언안에 정의된 스타일로 바꾼다.
· 웹 문서의 내용과 상관 없이 디자인만 다룸
· 따라서 다양한 기기에 맞는 탄력적 디자인 적용이 가능하고, 디자인 적용이 훨씬 편리해짐
· 선택자의 종류:
(앞에 아무것도 안 붙임)HTML요소 선택자(타입 선택자), (앞에 #붙임)id 선택자, (앞에 .붙임)class선택자, group 선택자가 있다. (여러 선택자를 ,로 묶어서 쓸 수 있음)

· CSS 적용법:
인라인 스타일: style 속성을 활용하여 태그에 직접 정의 스타일 편집이 번거로움
내부 스타일 시트: style 태그를 이용하여 head 태그 안에 정의
외부 스타일 시트: link 태그를 사용하여 head 태그 안에 정의, 외부의 스타일 시트 파일을 불러옴, 유지보수 편리

· 스타일 우선순위:
· 적용 범위에 따른
1. !important(예: 속성: 속성값 important!의 형태)
2.인라인 스타일(태그의 style 속성을 사용)
3. id 스타일(선택자로 #id를 사용)
4. 클래스 스타일(선택자로 .class를 사용)
5. 타입 스타일(선택자로 태그 이름 사용)

· 작성 순서에 따른
· 만약 CSS스타일 내부에서 같은 속성을 사용했다면, 나중에 적은 것이 적용됨(절차적)

· 중요도에 따른
1. 컴퓨터 사용자 스타일(예: 시각장애인용 고대비)
2. 웹 문서 제작자 스타일
3. 웹 브라우저 기본 스타일

· 스타일 상속:
· 태그 상속 관계에 따라 부모 태그(포함 하는 쪽의 태그) to 자식 태그(포함 당하는 쪽의 태그)
· 스타일 상속이란, 자식 요소에 스타일을 따로 지정하지 않으면, 부모의 스타일이 자식 요소에게 그대로 전달된다.
(예: body태그는 모든 요소의 부모 요소 이므로 body태그의 스타일은 모든 요소에 적용됨)
· background-color와 background-img는 스타일 상속이 안됨.

속성

· 속성, 속성값에 대한 설명은 W3C 표준 규약인 기본형 표기 방식을 사용
· 실제로 스타일 시트에 저렇게 작성하는 것이 아니라 속성과 속성값의 명세에서 사용한다.
· 보기 간편하고 설명이 효율적이다.
· 속성값에는 값과 유형 2가지가 있다.

· 절대 크기: 웹 브라우저에서의 지정된 크기
· 상대 크기: 부모 요소 글자 크기 기준 상대 크기
· 크기: 웹 브라우저와 상관 없이 직접 크기 지정
· 백분율: 부모 요소 글자 크기 기준 백분율 크기

· 크기 지정
· 키워드 사용 글자 크기 지정 (잘 안씀)
· xx-small < x-small < small < medium < large < x-large < xx-large

· 단위 사용 직접 지정
· 음수X(크기 이므로)

· <절대 크기> 유형

· <상대 크기> 유형
· em: 부모 요소의 글꼴의 대문자 'M' 너비(1em)기준 비율
· rem: root(<html>)에서 지정한 글자 크기(1rem)기준 지율

· <크기> 유형
· ex:사용 글꼴의 소문자 x 높이(1ex)기준 비율
· px: 모니터의 1픽셀(1px)기준 비율 (크기)
· pt: 포인트, 일반 문서에서 많이 사용 (크기)

· <백분율> 유형
· %: 부모 요소 글자 크기 기준 백분율(%) (백분율)

font-style: normal | italic | oblique

· 폰트의 이텔릭체, 디폴트 스타일
· 글자 크기는 'normal', 'italic', 'oblique' 중 한 가지를 가진다.
· normal: 디폴트 속성값일반 형태
· italic: 이텔릭체 (주로 사용)
· oblique: 이텔릭체

font-weight: normal | bold | bolder | lighter | 100 | 200 | ... | 800 | 900

· 폰트의 굵기 스타일
· normal: 디폴트 속성값보통
· bold: 굵게
· bolder: 원래보다 굵게
· lighter: 원래보다 가늘게
· 100 ~ 900: 400은 normal, 700은 bold에 해당함 숫자로 굵기를 세밀하게 조정 가능

텍스트 관련

· 텍스트와 글꼴은 같은 의미로 보일수 있는데 완전히 다르다.
· 글꼴 스타일: 글자체(폰트)에 대한 것
· 텍스트 스타일: 문자 하나, 단어, 문장에서 사용하는 스타일이다.

color: <색상>

· 글자의 색 스타일

· <색상> 유형
· #RRGGBB: 16진수 표현(0~9, a, b, c, d, e, f) 00~ff까지 RGB 표현 RGB자리 두수가 중복되면 한 수만 적어도 됨(예: #ddd)
· hsl(숫자%, 숫자%, 숫자%), hsla(〃): 숫자는 백분율, 숫자 순서대로 hsl(hsla의 경우 hsla)CSS3부터 가능, hue(색상), saturation(채도), lightness(명도)의 축약형 hsla의 a는 alpha(불투명도)
· rgb(숫자, 숫자, 숫자)와 rgba(〃): 숫자는 0(섞이지 않음) ~ 255(가득 섞임), 숫자 순서대로 RGB(rgba의 경우 rgba)를 의미
· 영문명: 일반적인 색깔이름을 영문으로 적는 것. red, green, blue 등...

text-align: start | end | left | right | center | justify | match-parent

· 문단(공통 성 있는 주제를 다루는 몇 줄 정도의 문장의 묶음)의 텍스트 정렬
· start: 현재 텍스트 줄의 시작 위치에 맞추어 문단 정렬
· end: 현재 텍스트 줄의 끝 위치에 맞추어 문단 정렬
· left: 왼쪽으로 문단 정렬
· right: 오른쪽으로 문단 정렬
· center: 가운데로 문단 정렬
· justify: 양쪽으로 맞추어 문단 정렬
· match-parent: 부모 요소를 따라 문단 정렬

line-height: <크기> ??

· 텍스트 줄의 간격, 텍스트 새로 정렬시 사용

· 🔗<크기> 유형
· 숫자px: 픽셀
· 숫자: 글자 크기(px)에 * 숫자 만큼 곱해짐(예: 12px * 2.0 = 24px의 줄 간격)
· 숫자%: 글자 크기(px)에 * 백분율 만큼 곱해짐(예: 12px * 200% = 24px의 줄 간격)

· 새로 중앙 정렬을 하려면 line-height의 px를 height과 같게 하면 됨.

· 텍스트에 줄 표시

· none: 텍스트에 줄 없음
· underline: 텍스트에 밑줄 표시
· overline: 텍스트 위에 밑줄 표시
· line-through: 텍스트 중간에 밑줄 표시(취소선 표시)

· 텍스트 그림자

· 세로 거리: 텍스트로부터 그림자까지 세로 거리 양수: 글자 아래쪽 그림자, 음수: 글자 위쪽 그림자(필수 유형)

박스 모델 관련

· 박스 모델: HTML의 요소는 박스형태로 되어있다. (요소 == 박스 모델)
· 박스 모델은 4개의 박스가 겹쳐 있다. content ⊂ padding ⊂ border ⊂ margin 이렇게 포개져있다.

· 웹 문서에 박스 모델이 나열되는 2가지 형태
· 웹 문서에서 요소를 나열 할 때 요소에 따라 나열 법이 달라진다.
· 블록 레벨 요소: 요소(박스 모델) 혼자 한 줄을 다 차지한다. (너비가 100%, 예: h1, div, p 등)
· 인라인 레벨 요소: 요소(박스 모델)의 content 영역 만큼만 차지한다. span태그의 경우를 예로 들면 텍스트만큼만 영역을 차지한다. (예: span, img, string 등)

width(혹은 height): <크기> | <백분율> | auto ??

· 박스 모델에서의 content 영역의 너비(width) 혹은 높이(height)를 지정

· <크기> 유형
· 숫자px
· 숫자em

· <백분율> 유형
· 숫자%: <body>즉, 웹 브라우저의 너비 혹은 높이 * %

· auto: 디폴트 속성값콘텐츠의 양에따라 너비 혹은 높이가 자동 결정

box-sizing: border-box | content-box ??

· width, height은 박스 모델의 content영역만의 너비, 높이를 나타낸다.
· 따라서, 여러 요소를 배치할 떄 border 까지 너비, 높이를 겹치지 않게 계산해야한다.
· 이를 쉽게 해줄수 있도록 box-sizing속성의 속성값으로 border-box를 지정할 수 있다.

· border-box: border 영역까지 포함 너빗값 지정
· content-box: 디폴트 속성값기존과 같이, content영역만 너비값 지정

box-shadow: <수평 거리> <수직 거리> <흐림 정도> <번짐 정도> <색상> inset

· 박스 모델에 그림자 효과를 부여

· <수평 거리> 유형 필수 유형

연결 선택자

· 아래 키워드 설명에서 '단계'라 함은 '포함 단계'의 준말임
· 하위 요소: 특정 요소 기준으로 한 단계 이상 밑에 있는 요소 혹은 요소들 모두를 '특정 요소의 하위 요소'라고 정의
· 상위 요소: 특정 요소 기준으로 한 단계 이상 위에 있는 요소 혹은 요소들 모두를 '특정 요소의 상위 요소'라고 정의

· 부모 요소: 특정 요소 기준으로 바로 한 단계 위에 있는 요소를 '특정 요소의 부모 요소' 라고 정의
· 자식 요소: 특정 요소 기준으로 바로 한 단계 밑에 있는 요소를 '특정 요소의 자식 요소'라고 정의
· 손자 요소: 특정 요소 기준으로 두 단계 이상 밑에 있는 요소 혹은 요소들 모두를 '특정 요소의 손자 요소'라고 정의

· 형제 요소: 특정 요소가 부모 요소가 같은 경우 '특정 요소는 형제 요소'라고 정의
· 형 요소: 형제 요소 중 먼저 나온 요소
· 동생 요소: 형제 요소 중 나중에 나온 요소

상위 요소 하위 요소

· 이렇게 작성 하면 하위 선택자(descendant-selector; 자손 손택자) 라고 한다.
· 상위 요소에게 포함된 하위 요소를 모두 선택하는 선택자이다.
· 하위 요소에 스타일이 적용된다.

부모 요소 > 자식 요소

· 이렇게 작성하면 자식 선택자(child selector) 라고한다.
· 자식 요소에 스타일이 적용된다.

요소1 + 요소2

· 이렇게 작성하면 인접 형제 선택자(adjacent selector) 라고 한다.
· 형 요소(요소1)의 바로 다음 동생 요소(요소2)에만 스타일이 적용된다.

요소1 ~ 요소2

· 이렇게 작성하면 형제 선택자(sibling selector) 라고 한다.
· 형 요소(요소1)의 다음 동생 요소(요소2)모두를 찾아서 스타일이 적용된다.

속성 선택자

[속성]과 [속성 = 값]

· '속성'을 가지는 것에 스타일 적용
· '속성' = '값'을 가지는 것에 스타일 적용
· 선택자가 있으면 그 선택자에 해당하는 요소 중 +, 없으면 모든 요소

· 예시
· a[href]: a 요소에서 href 속성을 가지는 것에 스타일 적용
· [target = _blank]: 모든 요소 중target = "_blank"를 가지는 것에 스타일

[속성 ~= 값]

· 속성' = '값'을 가지는 요소에 스타일 적용
· 선택자가 있으면 그 선택자에 해당하는 요소 중, 없으면 모든 요소

· 예시
· div[class ~= content]: div 요소 중 class = "content" 를 가지는 요소에 스타일 적용
· [class ~= content]: 모든 요소 중 class = "content" 를 가지는 요소에 스타일 적용

[속성 |= 값]

· '속성' = '값' 혹은 '속성 ='값-(하이픈)(다른 문자로 이어짐)'을 가지는 요소에 스타일 적용
· 선택자가 있으면 그 선택자에 해당하는 요소 중, 없으면 모든 요소

· 예시
· div[class |= content]: div 요소 중 class = "content" 혹은 class = "content-example" 를 가지는 것에 스타일 적용
· [class |= content]: 모든 요소 중class = "content" 혹은 class = "content-example" 를 가지는 것에 스타일 적용

[속성 ^= 값]

· '속성' = '값으로 시작'하는 요소에 스타일 적용
· 선택자가 있으면 그 선택자에 해당하는 요소 중, 없으면 모든 요소

· 예시
· div[class ^= content]: div 요소 중 class = "content" 혹은 class = "content로 시작"를 가지는 것에 스타일 적용
· [class ^= content]: 모든 요소 중class = "content" 혹은 class = "content로 시작"를 가지는 것에 스타일 적용

[속성 $= 값]

· '속성' = '값으로 끝'나는 요소에 스타일 적용
· 선택자가 있으면 그 선택자에 해당하는 요소 중, 없으면 모든 요소

· 예시
· div[class $= content]: div 요소 중 class = "content로 끝"나는 요소에 스타일 적용
· [class $= content]: 모든 요소 중 class = "content로 끝"나는 요소에 스타일 적용

[속성 *= 값]

· '속성' = '값의 일부가 일치하는' 요소에 스타일 적용
· 선택자가 있으면 그 선택자에 해당하는 요소 중, 없으면 모든 요소

· 예시
· div[class *= content]: div 요소 중 class의 속성값에 "content"라는 문자열이 일부 일치하는 요소에 스타일 적용
· [class *= content]: 모든 요소 중 class의 속성값에 "content"라는 문자열이 일부 일치하는 요소에 스타일 적용

가상 클래스와 가상 요소

· link, visited, hover, active는 링크로 메뉴를 만들 때 잘 사용된다.
· link > visited > hover > active 순으로 사용해야한다. 순서가 바뀌면 스타일이 제대로 적용되지 않는다.

:link

· 링크를 가지는 요소에서 방문 하지 않은 링크인 요소에 스타일 적용

:visited

· 링크를 가지는 요소에서 방문한 링크인 요소에 스타일 적용

:hover

· 요소에 마우스 포인터를 올려놓을 때 스타일 적용

:active

· 요소를 클릭하는 순간 스타일 적용

:focus

· 요소에 포커스가 가면 스타일 적용
· tab으로 입력 커서가 어떤 요소에 맞춰지거나, 마우스로 어떤 요소에 초점이 맞춰질 때

:target

· 요소에 앵커로 이동된 순간 스타일 적용
· 링크: 웹 문서에서 같은 사이트 다른 웹 문서 이동, 혹은 다른 사이트로 이동할 때 사용한다.
· 앵커: 웹 문서에서 다른 위치로 이동할 때 사용한다.

· 예시
· #explain:target: id = "explain"인 요소에 앵커로 이동된 순간 스타일 적용

:enabled, :disabled

· 요소의 사용 가능 여부에 따른 스타일 적용

· 예시
· textarea:disabled: <textarea>가 disabled 속성을 가질 경우 스타일 적용
· textarea:enabled: <textarea>가 사용 가능 상태일 경우 스타일 적용

:checked

· 라디오 박스, 체크 박스에서 checked 속성을 가질 경우 스타일 적용
· checked속성은 체크가 됬을 때 자동 부여됨

· 예시
· input:checked + label: input중(라디오 박스, 체크 박스 한정?) checked 속성을 가지면 바로 아래 동생 요소에 스타일 적용


:not()

· 괄호 안에 해당하는 요소를 제외한 요소에 스타일 적용

· 예시
· input:not([type = radio]): input 요소 중 type = radio 인 것을 빼고 스타일 적용

구조 가상 클래스
부모 선택자 :only-child

· 부모 요소의안에 자식 요소가 하나 뿐일 때 선택

부모 선택자 자식 선택자:only-type-of

· 부모 요소안에 자식 요소가 하나 뿐일 때 자식 선택자에 해당하는 요소 선택

부모 선택자 :first-child

· 부모 요소의 첫 번째 자식 요소 선택

부모 선택자 :last-child

· 부모 요소의 마지막 자식 요소 선택

부모 선택자 자식 선택자:first-of-type

· 부모 요소 안의 자식 요소 중 자식 선택자에 해당하는 첫 번째 요소 선택

부모 선택자 자식 선택자:last-of-type

· 부모 요소 안의 자식 요소 중 자식 선택자에 해당하는 마지막 요소 선택

부모 선택자 :nth-child(n)

· 부모 요소의 자식 요소들 중 n 번째 선택
· n에는 수식 사용 가능 밑에 나오는 나머지 구조 가상 클래스도 동일

· 예시
· .content :nth-child(2n+1); 홀수 번째 자식 요소를 선택

부모 선택자 :nth-last-child(n)

· 부모 요소의 자식 요소들 중 끝에서 n 번째 선택

부모 선택자 자식 선택자:nth-of-type(n)

· 부모 요소 안의 선택자에 해당하는 자식 요소 중 n 번째 선택

부모 선택자 선택자:nth-last-of-type(n)

· 부모 요소 안의 선택자에 해당하는 자식 요소 중 끝에서 n번째 선택

가상 요소

· 문서의 특정 부분에 가상으로 요소를 만든다.
· 그 부분에 스타일을 지정하기 위해 사용

::first-line, ::first-letter

· 요소의 첫 줄 혹은 첫 글자에 스타일 적용
· ::first-letter은 첫번 째 글자가 첫 번째 줄에 있어야함 만약 첫번 째 줄에 <br>이 있어서 줄이 없다면 적용 안됨

::before, ::after

· 요소의 앞 뒤에 스타일 적용
· 요소의 앞 뒤에 스타일 시트를 사용하여 텍스트를 넣을 때 사용
