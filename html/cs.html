Library, API, Framework 차이

· Library는 어떤 기능을 구현할때 자주쓰거나 유용한 기능을 모아놓은 집합 (예: C++의 STL)
· API는 어떤 프로그램과 내가 구현한 프로그램 사이의 기능을 구현하기 위한 인터페이스 (예: 홈페이지(내 프로그램)의 로그인(기능 구현)기능을 위한 firebase(웹 로그인, DB 프로그램)의 로그인 기능 구현 API)
· Framework는 어떤 기능을 구현하기위해 따라야하는 것 (예: 언리얼 엔진에서 캐릭터를 구현하려면 ACharacter클래스를 반드시 상속받는 프레임워크를 따라야한다. )
